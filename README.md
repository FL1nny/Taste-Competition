#计划
搭建网站
可创建房间
房主选择模式，可自定义或导入一张图片<作为待发歌曲对应条件 如：第一首歌：与xxx相关，将该图片放于网站正中间>

房间内每个玩家轮到自己回合通过网站内置歌曲库发送一首歌（内置一接口可提前选歌）,网站将对歌曲片段随机裁剪并播放

其余玩家可通过打字发送自己所猜的该歌歌名并以聊天气泡形式按顺序悬浮在对应玩家头像旁并设置计时器以秒为单位计算每个玩家发出某一答案的时间
（网站可判断哪个玩家最先答出标准答案，该回合发歌的玩家也内置一个接口可结束这一轮,指定某一玩家获得这轮胜利（答出某一歌曲的外号或别名）（注意：当有玩家觉得是黑幕可发动全民公投（F1全票打飞），限定技，每个玩家一局游戏仅可发动一次））
